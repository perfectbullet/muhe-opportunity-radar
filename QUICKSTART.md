# 🚀 快速启动指南

## 第一次使用

### 1️⃣ 确认环境配置

检查 `.env` 文件是否配置了 API 密钥：

```bash
# 至少需要配置一个 LLM 提供商
SILICONFLOW_API_KEY=your_key_here
# 或
DEEPSEEK_API_KEY=your_key_here
# 或
QWEN_API_KEY=your_key_here
```

### 2️⃣ 启动应用

**Windows 用户**：
```bash
start.bat
```

**Linux/Mac 用户**：
```bash
chmod +x start.sh
./start.sh
```

**或直接运行**：
```bash
# 激活虚拟环境
.venv\Scripts\activate  # Windows
source .venv/bin/activate  # Linux/Mac

# 启动应用
python app.py
```

### 3️⃣ 访问界面

浏览器打开：**http://localhost:7860**

## 界面使用

### 📝 单一视角分析

1. 在"单一视角分析"标签页
2. 输入投资材料（如：公司财报、新闻等）
3. 选择一位投资大师
4. 点击"开始分析"
5. 查看该投资者视角的分析结果

**示例材料**：
```
茅台酒业：
- 市值：2.3万亿
- PE：32倍
- ROE：30%
- 毛利率：91%
- 品牌护城河强，供不应求
- 风险：估值偏高，消费降级影响
```

### 🔄 多视角对比

1. 在"多视角对比"标签页
2. 输入投资材料
3. 勾选多位投资者（2-5位推荐）
4. 点击"开始对比分析"
5. 查看：
   - 每位投资者的独立分析
   - AI 生成的综合对比总结（共识、分歧、建议）

**推荐组合**：
- **价值投资组合**：巴菲特 + 格雷厄姆 + 约翰·聂夫
- **成长投资组合**：彼得·林奇 + 菲利普·费雪
- **全面评估组合**：巴菲特 + 林奇 + 索罗斯

### 📚 历史记录

1. 在"历史记录"标签页
2. 功能：
   - **最近记录**：查看最新分析
   - **投资者筛选**：只看特定投资者的分析
   - **关键词搜索**：搜索包含特定词的分析

### 📊 统计信息

查看你的分析统计：
- 总分析次数
- 各投资者使用频率
- 分析类型分布

## 常见问题

### ❓ MongoDB 连接失败？

**正常现象！** 应用会自动降级到无数据库模式，核心功能不受影响。

如需历史记录功能：
1. 安装 MongoDB Community Server
2. 启动 MongoDB 服务
3. 重启应用

### ❓ API 密钥错误？

检查 `.env` 文件：
- 密钥格式正确
- 没有多余空格
- 文件位于项目根目录

### ❓ 分析速度慢？

- 检查网络连接
- 尝试切换 LLM 提供商
- 减少材料长度

### ❓ 如何更换 LLM 提供商？

编辑 `app.py` 第 33 行：
```python
def init_analyzer(provider: str = "siliconflow"):  # 改为 deepseek, qwen 等
```
